# Markdown Live Preview

[Markdown Live Preview](https://markdownlivepreview.com/) is a tiny web tool to preview Markdown formatted text.

## Setup

```
$ make setup
```

## Build

```
$ make build
```

## Local Development

```
$ make dev
```
